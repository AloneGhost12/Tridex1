üíª Code Examples

Here's how easy it is to use the new popup system:

// Success message await tridexSuccess('Operation completed successfully!', 'Success'); // Error message await tridexError('Something went wrong. Please try again.', 'Error'); // Confirmation dialog const confirmed = await tridexConfirm( 'Are you sure you want to delete this item?', 'Delete Confirmation' ); if (confirmed) { // User clicked "Confirm" console.log('Item deleted'); } // Custom popup with multiple buttons const result = await TridexPopup.custom( 'Choose your action:', 'Custom Dialog', [ { text: 'Save', action: 'save', type: 'success' }, { text: 'Cancel', action: 'cancel', type: 'secondary' } ] );
